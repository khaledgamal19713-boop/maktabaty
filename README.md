# مشروع أجناس علل الحديث (AJNAS)

منصة معرفية تفاعلية متخصصة تهدف إلى دراسة وتصنيف "أجناس علل الحديث الشريف" وتيسيرها للباحثين والمحققين في قالب عصري وتفاعلي، مع أدوات متطورة للتحقيق العلمي المتقدم وتتبع العلل.

## وثائق المشروع
- [وثيقة عمل طبقة المكتبة (Library Layer Specification)](docs/LIBRARY_LAYER.md): تفصيل دقيق لطبقات التصفح والتحقيق والمحرر، مع شرح لفلسفة العمل التشاركي في مشروع أجناس علل الحديث.

## المميزات الرئيسية
- **دراسة أجناس العلل:** تصنيف وتحليل علل الحديث الشريف وتتبعها.
- **التصفح السينمائي:** واجهات مستخدم جذابة تهدف لتعزيز الاستيعاب المعرفي.
- **التحقيق التشاركي:** إتاحة الفرصة للباحثين للمساهمة في توثيق النصوص وفوارق النسخ.
- **المعارف الذكية:** استخدام الذكاء الاصطناعي لتحليل الأسانيد وتراجم الرواة.
- **التخريج الذكي:** أدوات مساعدة للعزو العلمي الدقيق والمؤتمت.

## التقنيات المستخدمة
- **Frontend:** HTML5, CSS3 (Custom Styles), JavaScript.
- **Backend/Database:** Supabase (PostgreSQL, Auth, Realtime).
- **External APIs:** Hadith API, Google Fonts.
